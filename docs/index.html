A label for developers to add to their sites, so they can promote themselves and their country.

ABOUT

By creating a simple label programmers, designers entepreneurs and startups can promote themselves and their countries. Join the movement!!!

When someone visits a site that uses the MWLI label he immediately reacts with a smile. Either if it's a product or a movement, the user feels happy and immediately supports the project! By clicking the label he is transfered to this site and has the ability to search for more projects that are part of our big community, thus spreading the word about them.

COMMUNITY

Our community is consisted of all our contributors. We support movements, help startups and organize meetups.

In order to support our community you can add our code to your site, contribute via Github or buy something from our store.

CODE

In case you want to add our code on your site please contact us, so we can add you to our list.

Our code and documentation is hosted on Github.
